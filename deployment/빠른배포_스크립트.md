# ⚡ 5분 빠른 배포 스크립트

가장 빠르게 배포하는 방법 (Railway 사용)

---

## 🚀 1단계: 사전 준비 (1분)

### 필수 계정 생성
1. **GitHub**: https://github.com (코드 저장)
2. **Railway**: https://railway.app (무료 호스팅)

---

## 📦 2단계: 백엔드 배포 (2분)

### A. GitHub Repository 생성

```bash
# 1. 백엔드 폴더로 이동
cd C:\ruffy\carbon_ai_ver3\react-agent

# 2. Git 초기화
git init
git add .
git commit -m "Backend initial commit"

# 3. GitHub에서 새 repository 생성
# Repository 이름: carbon-ai-backend
# Public 또는 Private 선택

# 4. GitHub에 푸시
git remote add origin https://github.com/[username]/carbon-ai-backend.git
git branch -M main
git push -u origin main
```

### B. Railway 배포

1. **Railway 로그인**: https://railway.app
2. **New Project** 클릭
3. **Deploy from GitHub repo** 선택
4. **carbon-ai-backend** 선택
5. **Environment Variables** 추가:
   ```
   ANTHROPIC_API_KEY=sk-ant-api03-...
   LANGSMITH_API_KEY=lsv2_pt_...
   NETZ_MCP_URL=https://hooxi.shinssy.com
   NETZ_MCP_ENABLED=true
   NETZ_ENTERPRISE_ID=1
   ```
6. 배포 완료 대기 (1-2분)
7. **URL 복사**: `https://carbon-ai-backend-production.up.railway.app`

---

## 🎨 3단계: 프론트엔드 배포 (2분)

### A. 환경 변수 수정

```bash
# 1. 프론트엔드 폴더로 이동
cd C:\ruffy\carbon_ai_ver3\agent-chat-ui

# 2. .env.production 수정
```

`.env.production` 파일 열기:
```env
NEXT_PUBLIC_API_URL=https://carbon-ai-backend-production.up.railway.app
NEXT_PUBLIC_ASSISTANT_ID=agent
```
→ 백엔드 URL을 방금 복사한 URL로 변경!

### B. GitHub Repository 생성

```bash
# 3. Git 초기화
git init
git add .
git commit -m "Frontend initial commit"

# 4. GitHub에서 새 repository 생성
# Repository 이름: carbon-ai-frontend

# 5. GitHub에 푸시
git remote add origin https://github.com/[username]/carbon-ai-frontend.git
git branch -M main
git push -u origin main
```

### C. Railway 배포

1. **Railway 대시보드**로 돌아가기
2. **New Project** 클릭
3. **Deploy from GitHub repo** 선택
4. **carbon-ai-frontend** 선택
5. Railway가 자동으로 Next.js 감지
6. **Environment Variables** 추가:
   ```
   NEXT_PUBLIC_API_URL=https://carbon-ai-backend-production.up.railway.app
   NEXT_PUBLIC_ASSISTANT_ID=agent
   ```
7. 배포 완료 대기 (2-3분)
8. **URL 복사**: `https://carbon-ai-frontend-production.up.railway.app`

## 🔧 문제 해결

### 백엔드 에러
1. Railway → Backend Project → Logs 확인
2. Environment Variables 확인 (API Key 오타?)

### 프론트엔드가 백엔드에 연결 안 됨
1. `.env.production`의 URL 확인
2. 백엔드 URL 끝에 `/` 없어야 함
3. Railway → Frontend Project → Redeploy

### 배포 실패
1. Railway Logs 확인
2. `requirements.txt` 또는 `package.json` 문제
3. Railway Discord에서 도움 요청
