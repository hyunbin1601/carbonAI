## 🎯 데이터 활용 계획

### 1. 지식베이스 구축
수집한 문서를 `react-agent/knowledge_base/` 폴더에 저장하고 벡터 DB 구축
```bash
cd react-agent
python build_vectorstore.py --force
```

### 2. RAG 시스템 최적화
- 실제 질문 패턴으로 검색 알고리즘 튜닝
- 키워드 추출 모델 개선
- 하이브리드 검색 가중치 조정

### 3. 프롬프트 개선
- 세그먼트별 맞춤 프롬프트
- FAQ 기반 답변 패턴 학습
- 프로세스 문서로 단계별 가이드 생성

### 4. 테스트 및 평가
- 수집한 질문으로 챗봇 테스트
