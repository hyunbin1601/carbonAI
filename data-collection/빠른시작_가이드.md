## 📋 수집할 것 (우선순위별)

### ⭐⭐⭐ 1-2주 내 (최우선)

1. **실제 사용자 질문 100개** (2-3시간)
   - 파일: `02_사용자질문_수집템플릿.md`
   - 또는: `사용자질문_수집양식.csv` (엑셀용)
   - 고객이 실제로 한 질문 그대로

2. **FAQ 50개** (3-4시간)
   - 파일: `03_FAQ_수집템플릿.md`
   - 질문 + 공식 답변 + 관련 질문

3. **주요 프로세스 5개** (5-10시간)
   - 파일: `06_프로세스문서_템플릿.md`
   - 배출권 구매/판매/보고 절차

### 🔹 3-4주 내 (중요)

4. **전문 용어 200개** (5-6시간)
   - 파일: `05_전문용어사전_템플릿.md`

   - KAU, KCU, Scope 등

5. **지식베이스 문서 10-20개** (지속적)
   - 파일: `04_지식베이스_체크리스트.md`
   - 배출권 가이드, 규제 문서 등

6. **고객 세그먼트 분석 6개** (6-12시간)
   - 파일: `07_고객세그먼트_분석템플릿.md`
   - 구매자, 판매자 등 유형별 특성

## 💡 작성 팁

### 실제 질문 수집
```
❌ "배출권 거래의 절차적 방법론에 대해 문의드립니다"
✅ "배출권 거래 어떻게 하나요?"
```
→ 고객이 실제 사용하는 표현 그대로

### FAQ 작성
```
질문: 배출권 거래는 어떻게 시작하나요?
답변: (200-500자로 구체적으로)
출처: 한국거래소 가이드
관련 질문: 3개 제시
```

### 프로세스 문서
```
1단계 → 2단계 → 3단계
각 단계마다:
- 목적
- 세부 작업
- 체크리스트
- 주의사항
```

