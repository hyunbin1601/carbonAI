# 실제 사용자 질문 수집 템플릿

**우선순위**: ⭐⭐⭐ 최우선
**목표 개수**: 최소 100개 (카테고리별 30개 이상)
**소요 시간**: 예상 2-3시간

---

## 📝 작성 가이드

### 수집 출처
- 과거 이메일 문의
- 전화 상담 녹취록
- 채팅 상담 로그
- 웹 폼 문의
- 고객 미팅 기록

### 작성 원칙
1. **원문 그대로**: 고객이 실제로 사용한 표현 그대로 기록
2. **다양한 표현**: 같은 의도라도 다양한 표현 방식 포함
3. **오타 포함 가능**: 실제 고객이 오타를 냈다면 그대로 기록
4. **문맥 포함**: 질문만으로 이해가 어렵다면 간단한 배경 설명 추가

## 📋 필드 설명

### 카테고리
- `탄소배출권`: 배출권 거래, 구매, 판매 관련
- `규제대응`: 규제, 보고, 컴플라이언스 관련
- `고객상담`: 서비스 안내, 일반 문의

### 고객_유형
- `배출권_보유자`: 배출권을 보유하고 있는 기업
- `배출권_구매자`: 배출권 구매를 원하는 기업
- `배출권_판매자`: 배출권 판매를 원하는 기업
- `배출권_생성_희망자`: 배출권 생성 프로젝트 추진
- `규제_대응_담당자`: 규제 대응 업무 담당
- `ESG_보고_담당자`: ESG 보고서 작성 담당
- `일반`: 분류되지 않는 일반 문의

### 질문_의도
- `정보_요청`: 단순 정보 제공 요청
- `절차_문의`: 프로세스/절차 문의
- `문제_해결`: 문제 상황에 대한 해결 방법 요청
- `상담_요청`: 1:1 상담 요청
- `개념_이해`: 개념/용어 설명 요청

### 질문_유형
- `what`: 무엇? (정의, 설명)
- `how-to`: 어떻게? (방법, 절차)
- `when`: 언제? (시기, 일정)
- `where`: 어디? (위치, 장소)
- `why`: 왜? (이유, 배경)
- `which`: 어느 것? (선택, 비교)

### 난이도
- `초급`: 기본 개념, 자주 묻는 질문
- `중급`: 실무 적용, 구체적인 절차
- `고급`: 복잡한 상황, 전문적 지식 필요

### 기대_답변_형식
- `단계별_가이드`: 1단계, 2단계... 형식
- `비교표`: 항목별 비교
- `체크리스트`: 필요한 것들 나열
- `가격_정보`: 금액, 비용 안내
- `절차_설명`: 프로세스 설명
- `계산_공식_및_예시`: 수식과 예시
- `일정_안내`: 날짜, 기한 안내
- `제재_내용`: 벌칙, 제재 내용
- `서비스_목록`: 서비스 리스트
- `맞춤_추천`: 상황별 추천
